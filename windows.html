Операційна система Windows Server 2022

Windows Server 2022 – це серверна операційна система від Microsoft, що є частиною сімейства Windows NT. Операційну систему було анонсовано 2 березня 2021 року.

Windows

Windows Server 2022 або Windows Server 2022 Edition — це десятий і останній випуск основного каналу довгострокового обслуговування (LTSC) операційної системи Windows Server від Microsoft у складі сімейства операційних систем Windows NT. Про це було оголошено на заході Microsoft Ignite з 2 по 4 березня 2021 року. Він був випущений 18 серпня 2021 року майже через три роки після Windows Server 2019 і за кілька місяців до Windows 11.Windows Server 2022 походить від кодової бази Windows 10 і, як і його попередники, сумісний лише з 64-розрядними процесорами.


Windows Server 2022 is derived from the Windows 10 codebase and, like its predecessors, is only compatible with 64-bit processors.


Історія

22 лютого 2021 року Microsoft оголосила, що Windows Server 2022 буде випущено 2 березня.


3 березня 2021 року корпорація Майкрософт оголосила про випуск Windows Server 2022 як попередньої збірки на Windows Update. Windows Server 2022 було запущено для загального доступу 18 серпня 2021 року.


У вересні 2021 року Microsoft оголосила, що випуск SQL Server 2022 заплановано на березень 2022 року.


У червні 2022 року корпорація Майкрософт випустила додаткові оновлення «C», щоб користувачі могли тестувати майбутні виправлення для Windows Server 2022 (KB5014665). Хоча ці оновлення вирішують проблеми з підключенням під час використання точок доступу Wi-Fi після встановлення оновлень Windows, також повідомлялося про проблеми з клієнтами LLTP/SSTP VPN і RDP, які не можуть підключитися після розгортання цих оновлень.


Можливості

    Безпека

  • TPM 2.0

  • Захищений базовий сервер; Credential Guard і Hypervisor-protected Code Integrity (HVCI).

  • Безпечне завантаження UEFI

  • Захист DMA при завантаженні

  • DNS через HTTPS

  • Шифрування AES-256 на SMB

    Зберігання

  • Служба міграції сховища

  • Стиснення блоку повідомлень сервера (SMB).

  • Безпека та продуктивність зберігання

    Хмара

  • Гібридні можливості Azure
